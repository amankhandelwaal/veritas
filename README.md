# Veritas
> Truth, on-chain.

A decentralized, censorship-resistant social platform built on Ethereum.
Posts are stored on IPFS, with content hashes committed to the Sepolia testnet.

## Stack
- **Contracts**: Solidity + Foundry (Sepolia Testnet)
- **Frontend + Backend**: Next.js (App Router) + Wagmi + RainbowKit
- **Storage**: IPFS via Pinata
- **Moderation**: Google Perspective API + DAO-lite smart contract voting

## Structure
- `/contracts` — Solidity smart contracts (Foundry)
- `/frontend` — Next.js app (UI + API routes)
